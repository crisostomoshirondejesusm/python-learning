# Exercises – Python Essentials 1

Esta pasta contém pequenos programas independentes, cada um resolvendo um exercício
específico de lógica e programação em Python.

## Scripts

### `loops_1.py`
Programa que lê vários números do usuário e mostra o maior valor utilizando um loop `while`.

### `lab_2_calculador_de_imposto.py`
Calcula o imposto de renda anual com base em faixas de rendimento e regras predefinidas.

### `lab_1.py`
Utiliza estruturas condicionais para verificar a entrada do usuário relacionada à palavra
"Spathiphyllum".

### 'comparando_numeros.py'
introducao a estruturas condicionais. utiliza estruturas condicionais para comparar 3 numeros fornecidos pelo usuario
e mostrar o maior na tela.
